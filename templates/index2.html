「HARUKAZE」の文法、使い方

「HARUKAZE」の文法は基本的に以下の命令を使って、 命令:処理　終　のように記述します。特にインデントや空白に規則はありませんが命令は一行に一つずつ記述することをおすすめします。入力コードは全角、半角どちらでもかまいませんが、　+ - / * = : ; < > { } などの記号及び数字は必ず半角で入力してください。（ただし文字列として入力する場合は全角でも可）

「HARUKAZE」の命令一覧

出力

文字列や数字を実行結果に表示します。C言語のprintf、Pythonのprint関数に似ています。(例　出力："春風" 終)

文字入力

入力欄に入力された文字を変数に代入します。(例　文字入力：言語(変数名) 終、(入力欄)　春風）

数値入力

入力欄に入力された数値を変数に代入します。(例　数値入力：回数(変数名) 終、(入力欄)　5）

計算

与えられた計算式を実行し数字を実行結果に表示します。(例　計算：2+3 終)

文字変数

与えられた文字列を変数に代入します。(例　文字変数:食べ物=りんご　終)

数値変数

与えられた数字を変数に代入します。(例　数値変数:数字=100　終)また計算命令を組み込むことによって変数を更新します。この場合最後の"終"の文字は必要なく代わりに計算命令を{}で囲んでください。（例　数字={計算：数字+2 終}）

分岐

; の前で与えられた条件が真であれば処理を実行します。最後の"終"の文字は必要なく処理の最後は}を付けてください。ｃ言語、pythonのif文に似ています。(例　分岐:数字==100;出力：春風　終})

繰り返し

; の前で与えられた回数だけ処理を繰り返します。最後の"終"の文字は必要なく処理の最後は}を付けてください。ｃ言語、pythonのfor文に似ています。(例　繰り返し：3;出力："春風"　終})

条件繰り返し

; の前で与えられた条件が真であれば処理を繰り返し実行します。最後の"終"の文字は必要なく処理の最後は}を付けてください。ｃ言語、pythonのwhile文に似ています。(例　条件繰り返し:数字<200;出力：春風　終 数値変数：数字={計算：数字+2 終}})

関数定義

命令を自分で定義します。定義した命令は　命令名: で実行できます。phpのfunction、pythonのdefに似ています。(例　関数定義:春風(命令名);出力：春風　終})
